Register a new bot
==================
POST /bots
    BODY: {
        email: string,
        name: string
    }
    RESPONSE:
        200 {botToken}
        409 existing name/email


Get bot info
============
GET /bots/{botToken}
    RESPONSE:
        200 {bot}
        404 invalid botToken


List boards
===========
GET /boards
    RESPONSE:
        200 Board[]


Join a specific board
=====================
POST /boards/{boardId}/join
    BODY: {
        botToken: string
    }
    RESPONSE:
        200 {botToken}
        403 Invalid botToken
        404 Invalid board
        409 Full board, Bot already joined


Perform a move on a specific board
==================================
PUT /boards/{boardId}/move
    BODY: {
        botToken: string,
        direction: Direction
    }
    RESPONSE:
        200 Board
        400 other error
        # -> Aktuellt fortf? 401 boardToken expired
        403 Invalid botToken
        404 Invalid board
        409 Invalid move


Get current board state
=======================
GET /boards/{boardId}
    RESPONSE:
        200 Board
        404 Invalid board


Get global highscore list
==================
GET /highscore
    RESPONSE:
        200 {highscore: Highscore[]}


Get stats
=========
GET /stats
    RESPONSE:
        200 Stats
