Register a new bot
==================
POST /bots
    BODY: {
        email: string,
        botName: string
    }
    RESPONSE:
        201 {botToken}
        400 existing botName/email/other


List boards
===========
GET /boards
    RESPONSE:
        200 {boardIds: [1]}


Join a specific board
=====================
POST /boards/{boardId}/join
    BODY: {
        botToken: string
    }
    RESPONSE:
        200 {boardToken}
        409 Full board


Perform a move on a specific board
==================================
PUT /boards/{boardId}/move
    HEADER:
        Authorization: Bearer {boardToken}
    BODY: {
        direction: Direction
    }
    RESPONSE:
        200 GameBoard
        400 other error
        401 boardToken expired
        404 Invalid board
        409 Invalid move


Get current board state
=======================
GET /boards/{boardId}
    HEADER:
        Authorization: Bearer {boardToken}
    RESPONSE:
        200 GameBoard
        401 Token expired


Get global highscore list
==================
GET /highscore
    RESPONSE:
        200 {highscore: Highscore[]}


Get stats
=========
GET /stats
    RESPONSE:
        200 Stats
